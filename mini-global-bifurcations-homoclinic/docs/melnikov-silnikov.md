# Melnikov Method and Shilnikov Analysis

## Melnikov Integral
M(t0) = integral_{-inf}^{inf} f(q0(t)) ^ g(q0(t), t+t0) dt

## Simple Zeros
When M(t0) has simple zeros, the stable and unstable manifolds intersect transversely. This is a necessary condition for chaos via the Smale-Birkhoff theorem.

## Shilnikov Saddle Quantity
sigma = |Re(lambda_stable)| / Re(lambda_unstable)
- sigma < 1: Shilnikov chaos
- sigma > 1: periodic attractor
- sigma = 1: bifurcation point

## Belyakov Transition
When sigma passes through 1, the system transitions from periodic to chaotic dynamics through a Shilnikov-Hopf bifurcation.
